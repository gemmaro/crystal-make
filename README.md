# make

NOTE: This library is *highly* experimental.

Rake-like task runner library.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     make:
       github: gemmaro/crystal-make
   ```

2. Run `shards install`
